Privacy Policy for Page Position Bookmarks

Last Updated: September 18, 2024
Quick Summary: Page Position Bookmarks stores your bookmark data locally on your device only. We don't collect, transmit, or share your data with anyone. You have full control over your data and can delete it at any time.

Overview

Page Position Bookmarks is a Chrome browser extension that allows users to save and jump to specific scroll positions on web pages. This privacy policy explains how we handle user data.

Data Collection and Storage

What Data We Collect

  • Bookmark Names: Custom names you assign to saved positions
  • Scroll Positions: Pixel coordinates of saved positions on web pages
  • URLs: Web page addresses where bookmarks are created
  • Timestamps: When bookmarks were created

How We Store Data

  • Local Storage Only: All data is stored locally on your device using Chrome's chrome.storage.local API
  • No External Servers: No data is transmitted to external servers or third parties
  • No Cloud Sync: Data remains on your device only

Data Usage

How We Use Your Data

  • Core Functionality: To save and restore your scroll positions
  • Bookmark Management: To display your saved bookmarks
  • Navigation: To jump to specific positions when requested

What We Don't Do

  • No Data Transmission: We never send your data to external servers
  • No Analytics: We don't track your browsing behavior
  • No Advertising: We don't use your data for advertising purposes
  • No Third-Party Sharing: We don't share your data with anyone

Data Security

Protection Measures

  • Local Storage: Data is encrypted by Chrome's storage system
  • No Network Access: Extension doesn't communicate with external servers
  • User Control: You can delete all data by removing bookmarks or uninstalling the extension

Your Rights

Data Control

  • View Data: All your bookmarks are visible in the extension popup
  • Delete Data: You can delete individual bookmarks or all data
  • Export Data: Data is stored in Chrome's local storage (accessible via developer tools)

Data Deletion

  • Individual Bookmarks: Delete specific bookmarks using the extension interface
  • All Data: Uninstall the extension to remove all stored data
  • Automatic Cleanup: Data is automatically removed when you uninstall the extension

Third-Party Services

No Third-Party Integration

  • No External APIs: We don't use any external services
  • No Data Sharing: We don't share data with third parties
  • No Tracking: We don't use tracking or analytics services

Updates to This Policy

Policy Changes

  • Notification: We'll update this policy if our data practices change
  • Version Control: Check the "Last Updated" date for changes
  • Continued Use: Using the extension after updates constitutes acceptance

Contact Information

Questions or Concerns

If you have questions about this privacy policy or how we handle your data, please contact us through the Chrome Web Store developer contact information.

Compliance

Chrome Web Store Requirements

This privacy policy complies with Chrome Web Store Developer Program Policies and provides comprehensive disclosure of our data practices as required.

Remember: Your data stays on your device. We don't collect, transmit, or share anything. You're in complete control.
